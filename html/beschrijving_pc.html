Website

Op deze pagina hebben we het over het bekijken en het aanmaken van websites.

Om een website te ontwerpen heb je een programma nodig dat je helpt bij het aanmaken van html webpagina's (html editor) én een programma om daarna die webpagina's te uploaden (FTP) op een server.

Het kan ook interessant zijn een programma te hebben dat een ganse website ineens kan downloaden.

De lokale kopies van de programma's kan je hier vinden.

Beschrijving: Bluegriffon is een Nederlandstalige WYSIWYG (what you see is what you get) html editor. Het programma is gebaseerd op Mozilla Firefox. Een html editor is pas nodig vanaf het tweede bachelorjaar indien je de optie elektronica-ICT kiest. Bluegriffon is zeer goed, maar de functionaliteit om de 'font size' (grootte van het lettertype) te veranderen is zeer omslachtig. Zeer handig is de mogelijkheid je html code te laten 'opkuisen'. Na wegschrijven is de html code zeer leesbaar in vb. kladblok.
