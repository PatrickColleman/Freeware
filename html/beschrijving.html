Beschrijving: Bluegriffon is een Nederlandstalige WYSIWYG (what you see is what you get) html editor. Het programma is gebaseerd op Mozilla Firefox. Een html editor is pas nodig vanaf het tweede bachelorjaar indien je de optie elektronica-ICT kiest. Bluegriffon is zeer goed, maar de functionaliteit om de 'font size' (grootte van het lettertype) te veranderen is zeer omslachtig. Zeer handig is de mogelijkheid je html code te laten 'opkuisen'.
