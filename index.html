Een omweg... .

 

Minstens één browser (Firefox in casu, maar misschien zijn er nog andere) maken problemen rond deze website. Je krijgt een melding te zien met een waarschuwing over de inhoud. Deze webpagina zou een "Gerapporteerde pagina met ongewenste software!" zijn, wat helemaal fout is natuurlijk! De reden hiervan is allicht omdat op deze website nogal wat 'executables' staan waarmee je programma's direct kan downloaden zonder de omweg naar de ontwerper van het programma. Daarom gebruiken we deze omweg (zie het in te vullen veld lager op deze pagina) om de webbrowser om de tuin te leiden. Hierdoor is er geen directe link meer naar de rest van de website.

Het doel van deze website is de gebruiker de weg te tonen naar nuttige freeware software. Freeware is gratis software die je legaal mag gebruiken. Alle software (programma's) waar deze website naar verwijst zijn geselecteerd op basis van de bronnen die o.a. onderaan deze pagina staan. De programma's die je kan downloaden zelf (de lokale kopies) zijn gecontroleerd door meerdere antivirus, anti spyware en anti adware programma's. De kans dat een programma 'verdacht' zou zijn is dus zeer klein. Een absolute waarborg kan natuurlijk nooit worden gegeven. De waarschuwing van de browser slaat dus hoogstwaarschijnlijk nergens op.
Personeel én studenten van de KULeuven kunnen hier ook een versie van Microsoft Office bekomen.
Eduroam en campusroam, de wifi omgevingen, kan je instellen met een tool die je op deze pagina kan downloaden.
Op deze pagina krijg je een overzicht van alle door de KULeuven aangeboden programma's (vb. Labview/Multisim, Matlab, Mathematica, Van Dale, ...) voor studenten (en personeel). Deze pagina bevat eveneens een link naar Microsoft Imagine (het vroegere Dreamspark), waar je ook veel software kan bekomen.

Gelieve, om verder te gaan, in het veld 'startpagina' hieronder de tekst start te typen en daarna op de knop 'Start' te klikken. Het ingevulde veld is onderdeel van de naam van de gevraagde webpagina. Typ het dus exact over of je krijgt een foutboodschap!
