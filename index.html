Wat uitleg... .

 

Wanneer je doorklikt na deze pagina, maakt minstens één browser (Firefox in casu, maar misschien zijn er nog andere) problemen rond dit deel van de website. Je krijgt een melding te zien met een waarschuwing over de inhoud. De volgende webpagina zou een "Gerapporteerde pagina met ongewenste software!" zijn, wat helemaal fout is natuurlijk! De reden hiervan is allicht omdat op deze website nogal wat links naar 'executables' staan waarmee je programma's direct kan downloaden zonder de omweg naar de ontwerper van het programma. De programma's die je kan downloaden zelf (de lokale kopies) zijn gecontroleerd door meerdere antivirus, anti spyware en anti adware programma's. De kans dat een programma 'verdacht' zou zijn is dus zeer klein. Een absolute waarborg kan natuurlijk nooit worden gegeven. De waarschuwing van de browser slaat dus hoogstwaarschijnlijk nergens op.

Deze website werkt zonder cookies. Er worden stiekem geen bestanden aangemaakt of informatie over de bezoekers bijgehouden!

Sinds mijn emeritaat staat deze website niet meer op een server van de KULeuven. Hij is verhuisd en staat nu op Github (voor de website zelf) en Google drive (voor de applicaties). Het nadeel hiervan is dat de reactiesnelheid lager kan zijn. Het gebruiksgemak is er echter nog steeds.

Het doel van deze website is de gebruiker de weg te tonen naar nuttige freeware software. Freeware is gratis software die je legaal mag gebruiken. Alle software (programma's) waar deze website naar verwijst zijn geselecteerd op basis van betrouwbare bronnen die o.a. onderaan deze pagina staan.

Personeel én studenten van de KULeuven kunnen hier ook een versie van Microsoft Office bekomen.
Eduroam en campusroam, de wifi omgevingen, kan je instellen met een tool die je op deze pagina kan downloaden. Op deze pagina krijg je een overzicht van alle door de KULeuven aangeboden programma's (vb. Labview/Multisim, Matlab, Mathematica, Van Dale, ...) voor studenten (en personeel). Deze pagina bevat eveneens een link naar Microsoft Azure (het vroegere Dreamspark), waar je ook veel Microsoft software kan bekomen.

Microsoft Office is op veel plaatsen verkrijgbaar, o.a. bij Microsoft zelf, Coolblue, Vanden Borre, Krefel, Media Markt, Exellent, bol.com, ... . Goedkopere oplossingen zijn o.a. licentiepromo, licentie2go, office-keys, productlicenties, gamekeydiscounter, cd-keys, beslist, licentie-sleutel, productsleutels, ... . Vergelijk zeker de prijzen!
