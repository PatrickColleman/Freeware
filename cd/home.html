Doel en wat uitleg... .

Het doel van deze website is de gebruiker de weg te tonen naar nuttige freeware software. Freeware is gratis software die je legaal mag gebruiken. Alle software (programma's) waar deze website naar verwijst zijn geselecteerd op basis van betrouwbare bronnen die o.a. onderaan deze pagina staan.

Browsers hebben de neiging foutmeldingen of strenge waarschuwingen te geven wanneer men tracht een pagina te openen met links naar executables. Het is net het doel van deze website de gebruikers te installeren programma's aan te bieden en installeerbare programma's zijn executables. Om deze neiging van browsers wat tegen te gaan, werden de executables verpakt in een zip bestand. Browsers zijn daar minder streng op. Het is dit zip bestand dat je eventueel kan downloaden als lokale link. Je kan eventueel het zip bestand gewoon openen als een map en de executable daaruit wegslepen naar een andere map.

Deze website werkt zonder cookies. Er worden geen stiekem bestanden aangemaakt of geen informatie over de bezoekers bijgehouden!

Als je vrij goed weet wat je nodig hebt, is de snelste manier in het linkerscherm bovenaan te klikken op 'Applijst'. Je krijgt dan een scherm met alle aanwezige programma's en een link met een beschrijving ervan. Je kan als alternatief ook de lijst van programma's of functionaliteiten gebruiken. Er verschijnt in beide gevallen dan een downloadpagina met verschillende min of meer samenhorende programma's, bijvoorbeeld antispam en e-mail. Een bepaalde downloadpagina zal dus verschijnen met meer dan één keuze in het menu links. Je kan dan voor elk programma de naam, de 'categorie', een waardeoordeel op een schaal van 5 .. 1 (5='need to have', 3='good to have', 1='nice to have'), een korte beschrijving en eventueel aanvullende informatie van/over het programma vinden.

Een aantal programma's bestaan in 32 bit en 64 bit versie. Gebruik de 64 bit versie als je een 64 bit versie van Windows hebt, dat is optimaler. Tegenwoordig zijn zo goed als alle versies van Windows 64 bit. Indien je niet weet welke versie je kan gebruiken, kies je voor de 32 bit versie. Om te onderzoeken welke Windows versie je gebruikt, volstaat het met de rechtermuistoets op 'deze computer' te klikken en de C schijf te openen. Indien je C schijf een map 'Program Files' én een map 'Program Files (x86)' heeft, heb je de 64 bit versie als operating systeem. Als lokale link bieden we alleen de 64 bit versie aan als die beschikbaar is.

Personeel én studenten van de KULeuven kunnen hier ook een versie van Microsoft Office bekomen.
Eduroam en campusroam, de wifi omgevingen, kan je instellen met een tool die je op deze pagina kan downloaden. Op deze pagina krijg je een overzicht van alle door de KULeuven aangeboden programma's (vb. Labview/Multisim, Matlab, Mathematica, Van Dale, ...) voor studenten (en personeel). Deze pagina bevat eveneens een link naar Microsoft Azure (het vroegere Dreamspark), waar je ook veel Microsoft software kan bekomen.

Microsoft Office is op veel plaatsen verkrijgbaar/te koop, o.a. bij Microsoft zelf, Coolblue, Vanden Borre, Krefel, Media Markt, Exellent, bol.com, ... . Goedkopere oplossingen zijn o.a. licentiepromo, licentie2go, office-keys, productlicenties, gamekeydiscounter, cd-keys, beslist, licentie-sleutel, productsleutels, ... . Vergelijk zeker de prijzen!
