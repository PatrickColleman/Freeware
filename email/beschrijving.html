E-mail en antispam

Op deze pagina hebben we het over een alternatief voor e-mail clients zoals Outlook én hoe men zich kan 'wapenen' tegen ongewenste spamberichten.

Indien je een webbrowser gebruikt om e-mails te lezen (webmail), is het niet nodig een aparte e-mailclient en/of antispamprogramma te installeren.

Vooreerst toch wat info over het configureren van je mailprogramma. IMAP (Internet Message Access Protocol) is een protocol om e-mails te ontvangen waarbij de e-mails op de server blijven staan (ideaal om vanaf meerdere computers aan je e-mail te kunnen). SMTP (Simple Mail Transfer Protocol) is een protocol is om e-mails te versturen. Je werkt dus met een combinatie van IMAP en SMTP. De KU Leuven ondersteunt het klassieke POP protocol (waarbij je e-mails niet op de server blijven staan) niet meer.

Eind 2023 migreerde de KULeuven voor personeelsleden (voor studenten gebeurde dit vroeger) de mailbox naar de Microsoft cloud. Dit maakt dat je mail en kalender het beste bereikbaar zijn via het Outlook programma (op Windows én Android). Op Android kan je eventueel FairEmail gebruiken. Thunderbird is eveneens bruikbaar. Beschrijving:
Blue Mail is een beetje een speciaal programma. Het is beschikbaar zowel in de Apple Store (voor IOS), Google Play (voor Android) als Microsoft Store (voor je Windows PC). Je kan het ook voor al die besturingssystemen downloaden op: https://bluemail.me/. Het voordeel tegenover Thunderbird is dat het een geïntegreerde agenda heeft. Het bevat, net zoals Thunderbird, geen reclame voor andere apps of producten. Bovendien heeft het een interessante functie om de instellingen van één computer naar een andere door te sturen. Blue Mail is niet meer bruikbaar om KULeuven mails in de Microsoft Cloud te lezen.
